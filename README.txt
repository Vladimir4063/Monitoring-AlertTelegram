BOT ALERT - Telegram - Python
We create queries to SQL Server and if the records have exceeded the parameters, an alert is created through telegram to attend to the case.

Creamos consultas a SQL Server y si los registros llevan superando los parametros, se crea una alerta atravez de telegram para atender el caso.

#Create VM
-virtualenv -p python3 env

#Activate
-env\Scripts>activate.bat

#Driver sqlServer
-pip install pyodbc

#Conexion a Api
-pip install requests
